한국에서 사용되는 행복복권 로또 번호 발행 앱을 안드로이드, 아이폰 버전으로 같이 만들고 싶어 시작부터 완성까지 액션플랜을 단계별로 나눠서 설명하고 만들어줘



요구사항을 정리할꺼야

앱에 들어가는 화면 
  - 스플래쉬 화면
  - 홈 화면
  - 번호 발행 탭 화면
  - 당첨 번호 리스트 탭 화면
  - 당첨 번호 통계 탭 화면
  - 북마크 번호 리스트 탭 화면
  - 설정 탭 화면

스플래쉬 화면 
  - 로고 화면을 3초건 보여주고 홈 화면으로 이동한다.

홈 화면
  - 하단 탭에 번호발행, 히스토리, 통계, 북마크, 설정 
  - 번호발행 탭 선택하면 번호 발행 탭 화면으로 이동한다.
  - 히스토리 탭 선택하면 당첨 번호 리스트 탭 화면으로 이동한다.
  - 통계 탭 선택하면 당첨 번호 통계 탭 화면으로 이동한다.
  - 북마크 탭 선택하면 북마크 번호 리스트 탭 화면으로 이동한다.
  - 설정화면 탭 선택하면 설정 탭 화면으로 이동한다.

각화면별 기능이야

스플래쉬 화면
 - 화면 중앙에 로고를 표시하는데 0.5초로 페이드인된다
 - 백그라운드에서 최신 당첨번호 기록을 확인하여 업데이트한다.
 - 업데이트 작업이 완료되면 홈화면으로 이동한다.

홈화면
 - 시작 탭은 번호 발행 탭이 시작화면이다.
 - 탭 하단에는 구글 띠 배너 광고가 표시된다.

번호 발행 탭 화면 
 - 추천번호, 높은확률, 낮은확률, 혼합선택 버튼들이 있다.
 - 추천번호는 5가지 방식으로 만들어진 번호들을 랜덤으로 5개 조합을 발행하는 기능이다.
 - 높은확률은 역대 최다 당첨 번호로 뽑힌 번호들을 가지고 랜덤으로 5개 조헙을 발행하는 기능이다
 - 낮은확률은 역대 최저 당첨 번호로 뽑힌 번호들을 가지고 랜덤으로 5개 조헙을 발행하는 기능이다
 - 혼합번호는 1~45번까지 6개까지 선택이 가능하며 선택을하지 않은 개수만큼 랜던으로 선택해서 6개 번호를 발행하는 기능이다.
 - 발행된 번호는 앞에 북마크 토글 버튼이 배치된다.
 - 북마크 토글 버튼을 누르면 저장된 번호가 아니라면 북마크DB에 번호와, 번호가 발행된 날짜시간이 저장된다.
 - 북마크 토글 버튼을 누르면 북마크로 설정된 번호는 해제되고 북마크DB에서 삭제된다.

당첨 번호 리스트 탭 화면
 - 역대 당첨번호들이 리스트로 표시된다.
 - 화면 상단에는 검색바가 위치하고있다
 - 검색바에 번호를 ','로 구분하여 입력하면 입력된 번호가 모두 포함된 당첨번호들이 리스트로 표시된다.
 - 리스트에 표시되는 ROW에 내용은 당첨번호, 회차, 추첨일자, 1등당첨금액이 표시된다.

당첨 번호 통계 탭 화면
 - 1~45번이 리스트형태로 표시된다
 - 리스트에 표시되는 ROW에 내용은 번호이미지, 가로형 막대그래프 표시, 횟수 표시
 - 가로형 막대그래프의 최소는 0이고 최대는 최고 많이 뽑힌 번호의 횟수에 올림한 값이다. 

북마크 번호 리스트 탭 화면
 - 북마크로 설정된 번호들이 리스트로 보여진다.
 - 리스트에 표시되는 ROW에 내용은 북마크 토글 버튼, 번호이미지들, 발급날짜
 - 북마크 토글 버튼을 누르면 저장된 번호가 아니라면 북마크DB에 번호와, 번호가 발행된 날짜시간이 저장된다.
 - 북마크 토글 버튼을 누르면 북마크로 설정된 번호는 해제되고 북마크DB에서 삭제된다.
 
설정 탭 화면
 - 로또 당첨번호 최신여부를 표시한다.
 - 앱 버전 정보를 표시한다.
